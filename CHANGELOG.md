# Changelog

All notable changes to `ui-sucks` will be documented in this file.

The format is intentionally simple and release-oriented. Package versioning is managed through Changesets.

## 0.1.0 - 2026-03-08

Initial public release.

### Added

- Headless `Button`, `Dialog`, and `Toast` primitives for React 18+.
- Strict TypeScript build with ESM, CJS, and type declaration output via `tsup`.
- Accessibility-focused keyboard interactions, ARIA wiring, and focus management.
- Vitest coverage for button and dialog behavior.
- Vite and Next.js example apps.
- CI workflow, bundle-size guardrails, and Changesets release tooling.

### Notes

- The package is intentionally minimal and style-agnostic.
- Additional primitives are documented in the roadmap for future releases.
