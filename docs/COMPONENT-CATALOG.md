# Component Catalog

This repository currently publishes three production primitives:

- `Button`
- `Dialog`
- `Toast`

The visual SVG atlas in the README presents the broader design direction for the library and its future primitive surface.

## Current Production Components

### Button

- Headless button primitive
- Optional toggle state
- Controlled and uncontrolled APIs

### Dialog

- Composable trigger/content/title/description primitives
- Modal and non-modal support
- Focus management and keyboard dismissal

### Toast

- Provider and viewport model
- Imperative enqueue API
- Auto-dismiss behavior with pause/resume interactions
