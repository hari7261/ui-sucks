# Roadmap

## Short Term

- Expand the primitive set around forms and overlays
- Add more component-level tests around focus and dismissal behavior
- Add benchmark snapshots for interaction-heavy primitives

## Medium Term

- Ship form building blocks such as `Input`, `Textarea`, `Select`, and `Combobox`
- Add menu primitives such as `DropdownMenu`, `Tooltip`, and `Popover`
- Add layout primitives such as `Tabs`, `Accordion`, and `BentoGrid`

## Long Term

- Introduce framework integration examples beyond Vite and Next.js
- Document composition recipes for design systems built on top of the library
- Add dedicated accessibility audits to CI
