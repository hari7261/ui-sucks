# Code of Conduct

## Our Standard

This project aims to be a respectful, technically rigorous, and welcoming place to collaborate.

Participants are expected to:

- Communicate professionally and respectfully
- Focus feedback on code, behavior, and outcomes
- Assume good intent while asking for clarity when needed
- Accept constructive review and correction

Unacceptable behavior includes:

- Harassment, threats, or personal attacks
- Discriminatory or degrading language
- Deliberate disruption of community discussions or reviews
- Publishing private information without consent

## Scope

This Code of Conduct applies across project spaces, including:

- GitHub issues
- Pull requests
- Discussions
- Release notes
- Community chat tied directly to the project

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer through the support channel documented in [SUPPORT.md](SUPPORT.md).
