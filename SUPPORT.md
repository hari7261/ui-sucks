# Support

## Getting Help

Use the following routes depending on the kind of help you need:

- Bug reports: open a GitHub issue
- Feature proposals: open a GitHub issue with a clear API proposal
- Security concerns: follow [SECURITY.md](SECURITY.md)
- General usage questions: use GitHub Discussions if enabled, otherwise open an issue

## Before Opening an Issue

- Reproduce the problem on the latest version
- Confirm whether the issue is in the library or in local styling/application code
- Include a minimal reproduction when possible
